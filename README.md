# 🌐 Лабораторная работа №3: Selenium-парсер и анализ сетевого трафика

> **Дисциплина:** Компьютерные сети  
> **Цель:** Написать скрипт-парсер с пагинацией и авторизацией, экспортировать данные в CSV, проанализировать различия HTTP/HTTPS.

---

## 📋 Описание
Проект реализует автоматизированный сбор данных с веб-сайта с помощью **Python + Selenium 4**. Скрипт:
- ✅ Обходит несколько страниц (пагинация)
- 🔐 Содержит готовый шаблон авторизации
- 📦 Сохраняет ≥4 параметров в `CSV`
- 🌍 Сопровождается теоретическим разбором HTTP vs HTTPS для отчёта

---

## 🛠 Требования
- Python `3.8+`
- `Selenium 4.x` (драйверы подтягиваются автоматически через Selenium Manager)
- Google Chrome / Chromium / Firefox
- Стабильное интернет-соединение

⚙️ Как это работает
Описание
🖥️ setup_driver()
Инициализирует Chrome с флагами обхода базовой защиты от ботов
🕵️ parse_page()
Ожидает загрузки элементов (WebDriverWait), извлекает текст, автора, теги, ссылку
🔄 handle_pagination()
Кликает Next, ждёт 2 сек, повторяет сбор. При отсутствии кнопки — останавливается
💾 save_to_csv()
Проверяет наличие данных, записывает в CSV с кодировкой utf-8-sig (совместимо с Excel)
🚪 finally
Гарантированно закрывает браузер даже при ошибках
